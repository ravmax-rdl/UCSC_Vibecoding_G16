# UCSC Vibe Coding – Group Project

A hands-on group project from the **University of Colombo School of Computing (UCSC)** exploring **vibe coding** combined with collaborative version control using **Git and GitHub**.

---

## Repository Structure

```
UCSC_Vibecoding_G16/
├── day_01/
│   ├── index.html           # Student ID card — Day 1 vibe coding output
│   └── reflection.md        # Prompts used and lessons learned
├── day_02/
│   ├── grade_calculator.py  # Student grade calculator with input validation and loop
│   ├── budget_tracker.py    # Monthly budget tracker with expense validation
│   └── reflection.md        # Prompts used and lessons learned
└── README.md
```
